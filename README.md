# 🎮 Simon Game

A classic Simon Memory Game built using **HTML**, **CSS**, and **Vanilla JavaScript**. The game tests the player's memory by generating an increasingly long sequence of colors that must be repeated correctly.

## 🚀 Features

* Interactive Simon memory game
* Random color sequence generation
* Sound effects for each button
* Button press animations
* Game over detection and restart functionality
* Clean and responsive user interface
* Built using modern Vanilla JavaScript (no frameworks)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/zayan-7909/Simon-game.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Press any key to start the game.

## 🎮 How to Play

1. Press any key to start.
2. Watch the sequence of flashing colors.
3. Repeat the sequence by clicking the buttons.
4. A new color is added after every successful level.
5. The game ends if you click the wrong button.
